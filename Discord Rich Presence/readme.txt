=====================================================================================
 REQUIREMENTS:
=====================================================================================

 This is a MWSE-Lua mod, and requires a valid installation of MWSE 2.1 or later.

    * Morrowind, Tribunal, and Bloodmoon.
    * MGE-XE 0.10.0
    * MWSE 2.1

=====================================================================================
 ABOUT THIS MOD:
=====================================================================================

 Discord provides a feature called Rich Presence. This allows more information to be
 displayed about the currently played game, and for games that support the features,
 allows you to join or spectate their games. This mod allows some extra data to be
 shown in Morrowind.

=====================================================================================
 INSTALLATION:
=====================================================================================

 Extract the archive into your Morrowind installation directory. The mod should
 contain the following files:
    .\Data Files\MWSE\mods\Discord Rich Presence\discord-rpc.dll
    .\Data Files\MWSE\mods\Discord Rich Presence\discordRPC.lua
    .\Data Files\MWSE\mods\Discord Rich Presence\main.lua

=====================================================================================
 PLAYING THIS PLUGIN:
=====================================================================================

 There is no esp file to activate. Simply install the MWSE 2.1 and this mod, and
 load up an old or new game.
